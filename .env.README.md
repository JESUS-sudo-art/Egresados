# Configuración de archivos .env

## Para desarrollo local (tu máquina)
Usa el archivo `.env.local`:
```bash
cp .env.local .env
php artisan config:clear
php artisan cache:clear
```

## Para servidor de producción
Usa el archivo `.env` (ya está configurado para producción)

## Diferencias principales

| Variable | Local (.env.local) | Servidor (.env) |
|----------|-------------------|-----------------|
| APP_ENV | local | production |
| APP_DEBUG | true | false |
| APP_URL | http://egresados.test | https://egresados.mesitest.com |
| MAIL_MAILER | log | smtp |

## Notas importantes
- **NUNCA** subas `.env` o `.env.local` al repositorio git
- El `.env` actual está configurado para servidor
- Para desarrollo local, siempre copia `.env.local` a `.env`
- Los correos en local se guardan en `storage/logs/laravel.log` (no se envían)
- Los correos en servidor se envían por Gmail SMTP
